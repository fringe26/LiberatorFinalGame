Fatty Sky - Scary. (2020/9/14 - by Fatty War, id3644@gmail.com)


This document explains how to use the pack.

Introduction
	It is a hand-painted sky asset suitable for various styles such as realistic, stylized and cartoon.
	There are 4 types of sky (Color variation - Blue, Green, Purple, Red).
	
	* Each sky is panorama type and box type.
	* It is a dark and cloudy sky without the sun.

included file contains the horizon at the origin.

1. File Description.

	1-1. File Naming : 
	
	* FattySky02 + Color = FattySky02_Blue
	* a+b.
	
	a : Package name : Fatty Sky - Scray.
	
	b : Variation.
		1. Blue.
		2. Green.
		3. Purple.
		4, Red.
	
	1-2. Materials
		*SkyBox (Assets\FattySkybox_Scray\Materials\SkyBox)
			-Box type sky material is here.

		*SkyPanoramic (Assets\FattySkybox_Scray\Materials\SkyPanoramic)
			-Panorama type sky material is here.

	1-3. Textures
		*SkyBox (Assets\FattySkybox_Scray\Textures\SkyBox)
			-Box type sky texture is here.
			-Sky box texture is 6 bundles.(front, back, left, right, top, down)

		*SkyPanoramic (Assets\FattySkybox_Scray\Textures\SkyPanoramic)
			-Panorama type sky texture is here.
